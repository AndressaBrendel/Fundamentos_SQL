
# Análise de Dados com fundamentos SQL | Vendas

Este projeto foi desenvolvido com o objetivo de consolidar e aplicar conceitos fundamentais de SQL voltados à análise de dados.

Atualmente curso Tecnologia em Sistemas para Internet e, ao longo da formação, tive contato com SQL em contexto acadêmico. Neste projeto, revisito esses conceitos com um novo direcionamento: **análise de dados aplicada a cenários de negócio**.

---

## Objetivo

Reforçar a base em SQL ao mesmo tempo em que desenvolvo o pensamento analítico, utilizando consultas para extrair insights a partir de uma base de vendas.

---

## Estrutura do Projeto

O projeto utiliza duas tabelas principais:

### `sales`
Contém informações de vendas:
- ID do pedido  
- Data  
- Produto  
- Categoria  
- Região  
- Quantidade  
- Valor da venda  
- ID do cliente  

### `customers`
Contém informações dos clientes:
- ID do cliente  
- Nome  
- Região  

A relação entre as tabelas permite análises mais completas utilizando JOIN.

---

## Conceitos trabalhados

Durante o desenvolvimento, foram aplicados os seguintes conceitos:

- SELECT
- WHERE
- ORDER BY
- DISTINCT
- Operadores lógicos (AND, OR, NOT)
- GROUP BY e funções de agregação (COUNT, SUM)
- JOIN (INNER JOIN)

---

## Análises realizadas

Mesmo com uma base de dados simplificada, foi possível responder perguntas relevantes, como:

- Quais produtos possuem maior volume de vendas  
- Quais clientes mais consomem  
- Qual o faturamento por cliente  
- Como os dados de diferentes tabelas se relacionam  
- Identificação de padrões a partir de filtros e agrupamentos  

---

## Aprendizados

Este projeto marcou uma transição importante na forma como utilizo SQL.

Deixando de lado o uso apenas técnico, passei a aplicar consultas com foco em **responder perguntas de negócio**, aproximando o estudo da realidade da área de dados.
